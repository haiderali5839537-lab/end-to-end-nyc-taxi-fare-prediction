NYC TAXI FARE PREDICTION: END-TO-END MACHINE LEARNING PROJECT

OVERVIEW
This project develops a complete end-to-end machine learning pipeline for predicting taxi fare amounts in New York City using historical NYC Yellow Taxi trip data. The workflow includes data preprocessing, feature engineering, exploratory data analysis, model development, hyperparameter tuning, and model interpretation.

PROBLEM STATEMENT
Given trip information such as pickup/drop-off locations, travel distance, passenger count, and pickup time, predict the corresponding taxi fare amount.

DATASET
NYC Yellow Taxi Trip Dataset

PROJECT WORKFLOW
1. Data Loading and Exploration
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Time-Aware Data Splitting
6. Linear Regression
7. Random Forest Regressor
8. Hyperparameter Tuning
9. Model Evaluation
10. Model Interpretation

TECHNOLOGIES USED
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

EVALUATION METRICS
- RMSE (Root Mean Squared Error)
- RMSLE (Root Mean Squared Logarithmic Error)

RESULTS
The Random Forest Regressor outperformed the baseline Linear Regression model and achieved the best predictive performance.
